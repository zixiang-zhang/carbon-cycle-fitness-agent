# 宏量营养素计算方法

## 基础代谢率 (BMR) 计算

### Mifflin-St Jeor 公式（推荐）

**男性：**
```
BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 + 5
```

**女性：**
```
BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 - 161
```

## 总每日能量消耗 (TDEE)

```
TDEE = BMR × 活动系数
```

| 活动级别 | 描述 | 系数 |
|---------|------|------|
| 久坐 | 办公室工作，无运动 | 1.2 |
| 轻度活动 | 轻度运动1-3天/周 | 1.375 |
| 中度活动 | 中等运动3-5天/周 | 1.55 |
| 高度活动 | 剧烈运动6-7天/周 | 1.725 |
| 极高活动 | 运动员/重体力劳动 | 1.9 |

## 碳循环各日类型热量

| 日类型 | 热量公式 | 示例(TDEE=2400) |
|--------|---------|----------------|
| 高碳日 | TDEE × 1.1 | 2640 kcal |
| 中碳日 | TDEE × 1.0 | 2400 kcal |
| 低碳日 | TDEE × 0.85 | 2040 kcal |

## 宏量营养素分配

### 蛋白质（优先计算）
- **推荐摄入**：1.8-2.2 g/kg 体重
- **热量转换**：1g 蛋白质 = 4 kcal
- **各日类型保持一致**

### 脂肪（设定下限）
- **最低摄入**：0.8 g/kg 体重
- **热量转换**：1g 脂肪 = 9 kcal
- **来源优选**：橄榄油、坚果、牛油果、深海鱼

### 碳水化合物（根据日类型调整）
- **热量转换**：1g 碳水 = 4 kcal
- **计算方式**：总热量 - 蛋白质热量 - 脂肪热量

## 计算示例

**用户信息：**
- 男性，30岁，身高175cm，体重75kg
- 中度活动（每周训练4天）
- 目标：减脂

**步骤1：计算BMR**
```
BMR = 10×75 + 6.25×175 - 5×30 + 5
    = 750 + 1093.75 - 150 + 5
    = 1698.75 kcal
```

**步骤2：计算TDEE**
```
TDEE = 1698.75 × 1.55 = 2633 kcal
```

**步骤3：各日类型热量**
- 高碳日：2633 × 1.1 = 2896 kcal
- 中碳日：2633 × 1.0 = 2633 kcal
- 低碳日：2633 × 0.85 = 2238 kcal

**步骤4：宏量分配（以高碳日为例）**
- 蛋白质：75 × 2 = 150g → 600 kcal
- 脂肪：75 × 0.9 = 67.5g → 608 kcal
- 碳水：(2896 - 600 - 608) / 4 = 422g

## 纤维素建议

- 每 1000 kcal 摄入约 14g 纤维
- 示例：2400 kcal 日 → 约 34g 纤维
- 来源：蔬菜、全谷物、豆类

## 水分摄入

- 基础：30-35 ml/kg 体重
- 运动日额外补充：运动时长 × 500ml
- 低碳日可适当增加以抑制饥饿感
